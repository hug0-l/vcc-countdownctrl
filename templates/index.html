📖 VCC PRE 操作說明 — 給廣播操作員

VCC PRE 是一套 網頁版廣播排控系統，讓您在同一畫面管理節目排程、設定 Cue 提示音、監控播出時序。所有資料自動儲存在您瀏覽器的 localStorage 中，重整或關閉頁面都不會遺失。

💡 建議首次使用時，點擊頂部列「⬇ 匯出備份」下載一次備份檔。之後所有修改都會自動儲存，無需手動操作。

📡 首頁 ─ 即時監控中心

點擊「🚨 啟動主控監測」按鈕即開啟 ENGINE：

  • 🚨 當前 Cue Point — 紅底面板顯示正在觸發中的提示點與節目名
  • ➔ 下一個 Cue Point — 黃底面板顯示即將到來的 Cue 與倒數秒數
  • Timeline 時間軸 — 每個節目一條軌道，半透明色塊 = 節目播出區間，│ 白色刻度 = Cue 點，▼ NOW 紅線隨時間移動
  • 矩陣表 — 以前近點為中心前後各 5 項，🔴 最接近 🟡 次接近 ⚪ 其餘。右側欄位顯示 Tags 與節目名稱
  • 點擊 ENGINE 按鈕可隨時停止 — 再次點擊重新啟動

📅 排程 ─ 節目管理

  • 行事曆 — 點日期切換觀測日。藍底 = 選取日，綠框 = 今天，紫底線 = 當日有排程，數字下方顯示節目數量
  • 輸入表單 — 填寫名稱、日期、開播時碼、長度、週期模式、Tags、顏色標籤後儲存
  • 🏷️ Tags 標籤 — 輸入文字按 Enter 或點「＋新增」，可過濾分類節目
  • 🎨 顏色標籤 — 點選色圈為節目指定專屬色（覆蓋自動編配）
  • 📆 本週縱覽 — 一週節目一覽，點擊節目方塊直接編輯
  • 動態節目表 — 點整行任一點進入編輯模式，排序點擊表頭

📋 Preset ─ 提示點設計

每個節目可套用一組 Preset，內含多個 Cue 節點：

  • 偏移方式 — 開始前／開始後／結束前／結束時，搭配 MM:SS 時間輸入
  • 提示音選取 — 從 9 種音色中選擇（短促、雙短、三短、長響、雙長響、急促 Alert、Chime、Bell、Tone）
  • 🔊 預聽按鈕 — 完整播放所選音色（含重複次數與間隔）
  • 拖拽 ⋮⋮ 排序節點；點表頭排序各欄
  • 底部 JSON 區 — 可直接編輯或匯入/匯出 Preset 定義
💡 開播/結束 cue 建議用高頻 (2000Hz)，提醒 cue 用中頻 (1000Hz)，背景 cue 用低頻 (400-800Hz)

🛠️ 設定 ─ 系統調校

  • 🔊 音訊振盪器 — 預設頻率與音長，變更自動儲存
  • FPS 格數 — PAL 25 / NTSC 29.97 / 電影 24fps。變更後 ENGINE 自動重算
  • 🌐 時區 — 設定運作時區（影響所有日期與 Cue 觸發時間）
  • 🗑️ 保留原則 — 已觸發 Cue 在矩陣中保留的秒數（5 秒預設）
  • ⬇ 匯出備份／⬆ 匯入備份 — 單一 JSON 檔包含所有設定、排程、Preset，可跨機器轉移
  • ⚙️ Config 面板 — 檢視/編輯原始設定 JSON

⚡ 快捷操作

  • ⏱ 一分鐘後 — 快速填入「現在 + 60 秒」的開播時碼
  • 點行編輯 — 動態節目表或本週縱覽點項目直接編輯，不必按 ✏️
  • Enter 新增標籤 — 在標籤輸入框按 Enter 快速加入
  • 拖曳 Preset 節點 — 抓 ⋮⋮ 排序
  • 表頭排序 — 點任一欄位表頭排序，再點切換升降序
  • ENGINE 切換 — 啟動後按鈕變為「點擊停止」，可隨時關閉 ENGINE
  • 🚨 CUE Popup — CUE 觸發時頁面頂部彈出醒目紅條提示音
  • 匯出/匯入 — 使用頂部列備份所有資料

⏱️ 時間與格數

時碼格式：HH:MM:SS:FF（FF = frame 格數）。預設 PAL 25fps（每幀 40ms），可在設定頁改為 29.97（NTSC）或 24（電影）。變更後需重新啟動 ENGINE 才會套用到 Cue 計算。

系統使用電腦本地時鐘配合指定時區運作。時區設定確保換日與 Cue 觸發時間正確。NTP 連線狀態僅供參考。

⚠️ 注意事項

  • 資料儲存在瀏覽器 localStorage，清除瀏覽資料前請先匯出備份
  • 啟用 ENGINE 後切換分頁會自動暫停 timer，返回時恢復
  • 「工廠重置」會清空所有資料（排程、Preset、設定），請先備份
  • 如需跨機器同步，使用頂部「⬇ 匯出備份」下載 .json，再到另一台匯入
  • NTP 校時依賴 worldtimeapi.org，無法連線不影響本地功能
